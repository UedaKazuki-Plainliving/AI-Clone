# User Clone Generator (コア利用者クローン自動生成スキル)

入力された任意のアンケートMarkdown情報から、製品検証や要求探索で活用可能な「コア利用者クローン（実ペルソナ）」を、プログラムおよび推論により完全に動的に生成します。

## 1. 情報の動的抽出ルール

入力されたアンケートMarkdownから、以下の項目を正確に抽出・マッピングします：
* **氏名**: ファイル名 `ヒアリング記入フォーム（[氏名]）.md` または「1. あなたについて」の署名から動的に抽出。
* **基本情報**: 年齢（歳）、性別、ふだん使う機器。
* **ITリテラシー値**: 「インターネットやアプリの操作の慣れ」のチェック位置から 1〜5 の数値で取得。
* **ペインポイント**: 「最悪だった・イラっとした出来事」「諦めていること」に書かれた具体的なテキストから、ユーザーの直面している最大の課題を抽出。
* **期待するニーズ**: 「まっ先に直したいこと」に書かれた具体的なテキストから、ユーザーの本質的なニーズを抽出。

## 2. 性格・品質メトリクスの動的計算式

アンケートの定量的回答から、以下の換算式に従って指標を動的に計算し、クローンのプロファイルに付与します。

### ■ TIPI性格5因子プロファイル (Ten-Item Personality Inventory)
アンケート設問24〜33の回答値（1〜7の整数）から、以下の式で5因子のスコア（1.0〜7.0）を動的に算出します：
* **外向性 (Extraversion)**: `(Q24 + (8 - Q29)) / 2`
* **協調性 (Agreeableness)**: `((8 - Q25) + Q30) / 2`
* **勤勉性 (Conscientiousness)**: `(Q26 + (8 - Q31)) / 2`
* **情緒安定性 (Emotional Stability)**: `((8 - Q27) + Q32) / 2`
* **開放性 (Openness to Experience)**: `(Q28 + (8 - Q33)) / 2`

### ■ システムユーザビリティスケール (SUS: System Usability Scale)
アンケート設問14〜23の回答値（1〜5の整数）から、以下の式で使いやすさスコア（0〜100点）を動的に計算します：
* `SUSスコア = ( (Q14-1) + (5-Q15) + (Q16-1) + (5-Q17) + (Q18-1) + (5-Q19) + (Q20-1) + (5-Q21) + (Q22-1) + (5-Q23) ) * 2.5`

### ■ 狩野モデルによる要求機能の分類 (Kano Model)
機能A〜Cに対する「もしあったら（正質問）」「もしなかったら（逆質問）」の回答（①〜⑤）から、以下の分類マトリクスを用いて、魅力品質 (A)、一元品質 (O)、当たり前品質 (M)、無関心品質 (I)、逆品質 (R)、疑問（Q）に動的に分類します。
* ①うれしい, ②あって当然, ③なんとも思わない, ④しかたない, ⑤こまる／いやだ

| 正 \ 逆 | ①うれしい | ②あって当然 | ③なんとも思わない | ④しかたない | ⑤こまる／いやだ |
|---|---|---|---|---|---|
| **①うれしい** | Q | A | A | A | O |
| **②あって当然** | R | I | I | I | M |
| **③なんとも思わない**| R | I | I | I | M |
| **④しかたない** | R | I | I | I | M |
| **⑤こまる／いやだ** | R | R | R | R | Q |

## 3. クローン出力形式

生成されたクローンは、以下の統一されたMarkdownフォーマットで出力します。

```markdown
### クローンID: user_[N] - [氏名]さん (年齢: [年齢] / 性別: [性別])
- **ITリテラシー値**: [1〜5]/5 ([リテラシー評価テキスト])
- **SUSスコア**: [計算された数値]点 (CSAT満足度: [設問12の値], NPS推奨度: [設問13の値])
- **TIPI性格プロファイル**:
  - 外向性: [スコア] / 協調性: [スコア] / 勤勉性: [スコア] / 情緒安定性: [スコア] / 開放性: [スコア]
- **機能品質分類 (Kano)**:
  - 機能A: [分類] / 機能B: [分類] / 機能C: [分類]
- **ペインポイント**:
  - [抽出された具体的なイライラや困りごと]
- **本質的ニーズ**:
  - [抽出された期待する解決策]
- **心理的・業務的葛藤のトリガー**:
  - [TIPI性格とペインから導き出される、対話で揺さぶるべき本人の恐怖・こだわり・プライドなどの心理的葛藤要因]
```
